## File Naming Convention

All sign videos follow the pattern:
```
sign_<word>.mp4          → single word (e.g., sign_dog.mp4)
sign_<word1>_<word2>.mp4 → compound phrase (e.g., sign_good_morning.mp4)
```

Words without a matching video are **fingerspelled** letter-by-letter using the alphabet sign videos.
